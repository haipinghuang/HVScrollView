# HVScrollView
仿同花顺自选股、持仓股多列可横向滚动的ListView,可支持松开手指后自动滑动到临界点。

![图片名称](https://github.com/haipinghuang/HVScrollView/blob/master/ext/2.gif)
# 功能
* 支持设定每列的列头内容
* 支持设定第一列（即固定列）的宽度
* 支持设定默认显示可以滚动的列数
* 支持设定是否松开手指后自动滑动到临界点
* 支持设定自定义ListView adapter
